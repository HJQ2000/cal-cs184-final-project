Problem Description

In project 3-1 and 3-2 we implemented pathtracing rendering on glass, mirror and microfacet material. However, we didn't have support for various textures. In this final project, we want to add texture support to make the rendering more appealing and interesting.

Goals and Deliverables

What we plan to deliver?

Our goal is to be able to map various textures including cloths and simple 2D textures to the object.

We want to keep the rendering fast, reusable, and can cope with different texture realistically. We think that the basic logic of implementing texture mapping is the same as the one introduced in class. We could build on that and try to find suitable reflection and refraction for varied texture with complicated texture such as clothes involves more complex reflection and refraction.

What we hope to deliver?

We would hope to achieve that having a rabbit wearing realistic cloth (including pleats and yarn) or make the rabbit looks hairy, and to accomplish this we might need to implement more than simple texture rendering since the pleats and shapes is not 2D but other 3D objects that's not part of the rabbit. If we have time, we would also like to try speeding up the process as more as possible.

Schedule

Week1: Implement feasible basic logic

Week2: Implement higher-level logic. The goal is to be able to render an image that at least looks similar to what we expect

Week3: Testing and further improving the algorithm. Make sure that we accomplishes the "what we plan to deliver?" part. Try to accomplish "what we hope to deliver?"

Week4: Work on final report & continue to work on "what we hope to deliver?" part
