Part 1: Mirror and Glass Materials

With max_ray_depth=0, we can only see the light source in the top of the image.
With max_ray_depth=1, there is no reflection nor refraction since light don't bounce and there is only direct lighting.
With max_ray_depth=2, we can see reflection in both balls but almost no refraction in the right ball. That is because the R value is large so that there is more reflection than refraction in the right-side glass ball.
With max_ray_depth=3, we can see both reflection and refraction in the image. However, since the light only bounces at most three times, in the left glass ball we can see that the reflected glass ball in the reflection is dark.
With max_ray_depth=4, the reflected glass ball in the reflection is no longer dark because we added one bounce to the rays.
With max_ray_depth=5, the image is similar to that with max_ray_depth=4, the only difference is that the image is slightly brighter due to the fact that the ray bounces one more time in the image.
With max_ray_depth=100, there is no big difference with the rendered image with max_ray_depth=5 because light bounces are killed due to Russian Roulette.



Part 2: Microfacet BRDF

As the alpha increase, the dragon is less reflective and have less noise. When alpha is small, we can see very strong relfection of the red and purple wall on the dragon surface. When alpha increase to 0.5, we see nearly no reflection but purely diffuse.



Bunny generating with cosine hemisphere sampling has more noise and black dots while bunny with importance sampling is more realistic. When using cosine hemisphere sampling, it samples randomly and thus result in more noise.



The bunny on the top is iron. The bunny on the bottom is cesium.
